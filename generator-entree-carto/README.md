# [PROTOTYPE] Generation du JSON de l'entré carto

Usage

```bash
python3 generator-entree-carto/entree-carto.py --input=dist/customConfig.json --output=entreeCarto_tmp.json 
```

On peut limiter les données à generer via le paramètre `count` :

```bash
python generator-entree-carto/entree-carto.py --input=dist/fullConfig.json --output=entreeCarto_tmp.json --count=100
```
